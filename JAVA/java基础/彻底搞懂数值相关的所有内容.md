现在的关键问题是，为什么负数的绝对值比正数的大 1？

https://www.jianshu.com/p/35cf507ebe7f

这篇文章通过较为简单的 8位 Byte来说明问题，通过一种以小见大的 数学归纳法的思维方式，很简单的解释了原理。

大的那个1是因为0只需要一种状态来表示，而不需要区分所谓的正0和负0

所以 例如8位可以表示 256个状态

​	127 + 1 + 128 = 256
